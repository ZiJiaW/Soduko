此程序为BUAA软工个人项目，解决数独问题，可执行文件在BIN文件下，支持下列命令行参数：
-c N
生成N个不重复数独终局至同目录下文件sudoku.txt中。
-s path_of_file
解绝对路径下文件中的数独，将结果输出值同目录下文件sudoku.txt文件中。
